ToDo
App.

A clean, functional task manager built with Django and PostgreSQL. Organise daily responsibilities under custom Work Headings with auto-cascade cleanup and a responsive UI.

View on GitHub Back

Key Features

01

Work Headings (Categories)

Create specific groups for different projects or life areas — e.g., "Personal", "Django Project", "Office" — to keep tasks neatly segmented.

02

Task Management

Add tasks with titles and descriptions directly into specific headings for clear, structured organisation.

03

Automated Cascade Cleanup

Database-level cascading means deleting a Work Heading automatically removes all associated tasks — no manual cleanup needed.

04

Simple Completion Flow

No complex status toggles — simply delete a task once completed for a friction-free productivity experience.

05

Modern Responsive UI

Styled with a clean, responsive CSS layout ensuring a consistent experience across all screen sizes.

Project Architecture

static/

Custom CSS styling for the entire application.

templates/

HTML layouts — category list, task list, task form, and delete confirmation.

todolist/

Django project config — settings.py and URL routing.

todo/

Core Django app — models, views, and forms for all task/heading logic.
